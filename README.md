# SubtitleTranslate

SubtitleTranslate旨在为用户提供字幕文本翻译功能，对常见字幕文件格式进行在线API翻译至用户指定语言文本。

## 软件功能

- 常见字幕文件格式支持
- 常见字幕文件格式转换
- 在线翻译API调用

## 立项初衷

本项目源自于作者日常需求，在观影/番时部分资源仅存在繁体中文字幕文件，而目前能够达成字幕翻译功能的软件，例如：Subtitle Edit等软件在使用过程中均存在一定的局限性或过于臃肿，所以基于该需求，作者决定立项开发该软件。

## 项目支持

- 编程语言：Python
- 翻译API：火山翻译
- AI辅助：TraeAI
